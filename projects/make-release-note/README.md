## 개요

**MakeReleaseNote**는 케이블 TV 셋톱박스 소프트웨어 릴리즈 노트를 PDF 형식으로 자동 생성하는 Java 데스크톱 애플리케이션입니다.

이전 릴리즈 노트 PDF를 읽어 버전 정보를 자동으로 파싱하고, 새로운 변경사항을 추가하여 표준화된 형식의 릴리즈 노트를 생성합니다.

---

## 주요 기능

### PDF 생성
- **iText 라이브러리**: 표준화된 릴리즈 노트 PDF 생성
- **한글 폰트 지원**: 나눔고딕 폰트 내장
- **자동 버전 증가**: MW/Glue 버전 자동 관리

### 자동화
- **PDF 파싱**: 이전 릴리즈 노트에서 정보 자동 추출
- **드래그 앤 드롭**: 파일 드래그로 간편한 로드
- **파일명 자동 생성**: 버전 기반 파일명 자동 설정

### 다중 사이트 지원
- TBroad, CJ, JEJU, CCS, HCN, CNM
- 사이트별 템플릿 적용

## 해결한 문제

### 1. PDF 텍스트 파싱
**문제**: 기존 PDF에서 버전 정보와 변경사항을 정확히 추출해야 함

**해결**: iText의 PdfTextExtractor를 활용하여 텍스트 추출 후, 정규 패턴 기반 파싱 로직 구현

### 2. 한글 PDF 생성
**문제**: 기본 폰트로는 한글이 깨지는 문제

**해결**: 나눔고딕 TTF 폰트를 JAR에 내장하여 한글 렌더링 지원

### 3. 버전 관리 자동화
**문제**: 수동 버전 관리로 인한 휴먼 에러 발생

**해결**: 파일명과 PDF 본문에서 버전을 파싱하여 자동 증가 및 일관성 검증
