## 개요

**BatteryAgent**는 macOS 메뉴바에서 동작하는 배터리 충전 제한 관리 앱입니다. SMC를 통해 배터리 충전 한도를 20~100% 범위로 제어하며, 14일간의 사용 패턴을 학습하는 스마트 충전, 캘린더 연동, 온도 기반 보호, 주간 리포트, Claude AI 분석 등 배터리 수명을 극대화하기 위한 종합적인 기능을 제공합니다.

---

## 주요 기능

### 배터리 충전 제한 제어
- 20~100% 범위에서 충전 한도 설정
- SMC(System Management Controller)를 통한 직접적인 충전 제어
- Helper Daemon을 활용한 권한 분리 아키텍처

### 스마트 충전
- 14일간 사용 패턴을 학습하여 최적의 충전 시점 결정
- 사용자의 일상 패턴에 맞춘 자동 충전 관리

### 캘린더 연동
- 일정 기반 충전 스케줄링
- 외출/미팅 전 자동 완충 준비

### 온도 기반 보호
- 배터리 온도 실시간 모니터링
- 임계 온도 초과 시 자동 충전 중단으로 배터리 보호

### 주간 리포트
- 배터리 사용 현황 주간 분석
- 충전 패턴 및 배터리 건강 상태 추적

### AI 분석 (Claude)
- Claude AI를 활용한 배터리 사용 패턴 분석
- 개인화된 배터리 관리 권장 사항 제공

## 개발 과정에서의 도전과 해결

### 1. SMC 배터리 충전 제어
**도전**: macOS에서 배터리 충전 한도를 직접 제어하려면 SMC(System Management Controller)에 접근해야 하며, 이는 루트 권한이 필요한 저수준 시스템 작업입니다.

**해결**: Helper Daemon을 통한 권한 분리 아키텍처를 설계하여, 메인 앱은 사용자 권한으로 동작하고 SMC 접근이 필요한 작업만 Privileged Helper를 통해 처리하도록 구현했습니다.

### 2. 스마트 충전 패턴 학습
**도전**: 사용자의 충전 패턴을 학습하여 최적의 충전 시점을 예측하는 알고리즘을 구현해야 했으며, 14일이라는 학습 기간 동안 충분한 데이터를 수집하면서도 빠르게 적응해야 했습니다.

**해결**: 시간대별 사용 패턴을 통계적으로 분석하고, 가중 이동 평균을 활용하여 최근 패턴에 더 높은 가중치를 부여하는 적응형 학습 알고리즘을 구현했습니다.

### 3. 온도 기반 안전 제어
**도전**: 배터리 온도를 실시간으로 모니터링하면서 시스템 리소스를 최소화하고, 임계치 초과 시 즉각적으로 충전을 중단해야 했습니다.

**해결**: 효율적인 폴링 전략과 임계값 기반 이벤트 시스템을 구현하여 배터리 온도 상승 시 단계적으로 대응하도록 설계했습니다.
