## Overview

**tvOS Player Sample** is a video player sample application for Apple TV (tvOS) platform.

It provides a reference implementation for AVFoundation-based video playback and InApp TV service integration.

---

## Key Features

### Video Playback
- **AVPlayer Integration**: tvOS native player
- **HLS Streaming**: HTTP Live Streaming support
- **DRM Support**: FairPlay Streaming compatible

### UI/UX
- **tvOS Focus Engine**: Remote navigation support
- **Fullscreen Playback**: Optimized viewing experience
- **Playback Controls**: Pause, seek, volume control
