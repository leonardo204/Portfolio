## 개요

**RClient ICS**는 셋톱박스에서 클라우드 기반 UI 이미지를 실시간으로 수신하고 렌더링하는 C 언어 클라이언트 라이브러리입니다.

AES 암호화 통신, 이미지 프레임 캐싱, 애니메이션 효과 등을 지원하며, 다양한 임베디드 플랫폼에서 동작합니다.

---

## 주요 기능

### 이미지 스트리밍
- **실시간 프레임 수신**: 서버에서 이미지 프레임 스트리밍
- **프레임 캐싱**: 효율적인 버퍼 관리
- **셀 기반 업데이트**: 변경된 영역만 선택적 업데이트

### 보안
- **AES 암호화**: AES-128/192/256 지원
- **암호화 통신**: 이미지 데이터 암호화 전송

### 애니메이션
- **전환 효과**: 페이드, 슬라이드 등 애니메이션
- **컴포지션**: 다중 레이어 이미지 합성

### 이미지 처리
- **다중 포맷**: PNG, JPEG, WebP 지원
- **디코딩 최적화**: 하드웨어 가속 활용

## 해결한 문제

### 1. 대역폭 최적화
**문제**: 전체 화면 이미지 전송 시 대역폭 부족

**해결**: 셀 기반 차분 업데이트로 변경된 영역만 전송, 대역폭 사용량 80% 감소

### 2. 보안 통신
**문제**: 이미지 데이터 도청 위험

**해결**: AES 암호화 적용으로 안전한 이미지 데이터 전송

### 3. 다중 플랫폼 지원
**문제**: 다양한 셋톱박스 플랫폼 지원 필요

**해결**: 플랫폼 추상화 레이어 설계로 Linux, Android, Tizen, webOS 지원
